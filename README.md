## Run Locally

Clone the project

```bash
  git clone https://github.com/Susmita549/adaptable-oven-8035.git
```

Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the localhost server

```bash
  npm start
```
